# TRUST BANK API

## Overview

- This is a bank account management api

## Dependencies

- MongoDB
- Mongoose

## Backend

    - API Documentation :
    - Language : Nodejs (Xpress framework)

## Feature List

- [x] Bank Account

## Database

    kind: MongoDB Atlas
    db-name : trust-bank-api

## Setting up for Development

- Clone project.
- Install Node ( >= 14.0.0).
- Install yarn/npm.
- Copy .env.example to .env file
- Run yarn/npm install.
- Run yarn/npm start.

## Execute
- yarn/npm start 
- yarn/npm run dev

## Test
```
yarn/npm run test
```
